# Screenshots Guide

Use this folder for README screenshots.

## Recommended Files
- `diag-ok.png`: `/助手 诊断` healthy output.
- `diag-auth-failed.png`: diagnosis showing auth failure.
- `help-menu.png`: `/助手 帮助` command list.
- `high-risk-confirm.png`: confirmation flow for high-risk action.

## Capture Rules
- Remove or blur personal IDs, tokens, filesystem secrets, and hostnames if sensitive.
- Keep UI language consistent (Chinese recommended for this project).
- Prefer PNG, width >= 1280px for readability.
